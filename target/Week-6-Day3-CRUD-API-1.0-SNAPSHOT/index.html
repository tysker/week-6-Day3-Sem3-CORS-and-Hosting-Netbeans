Week 6 CRUD API



Enter the following:

Get all Person from the List: http://localhost:8084/Week-6-Day3-CRUD-API/api/crud

Get Person by Id: http://localhost:8084/Week-6-Day3-CRUD-API/api/crud + Persons Id

Add Person: http://localhost:8084/Week-6-Day3-CRUD-API/api/crud/add

Delete Person by Id: http://localhost:8084/Week-6-Day3-CRUD-API/api/crud/del + Persons Id

Update Person: http://localhost:8084/Week-6-Day3-CRUD-API/api/crud/update + Persons Id
